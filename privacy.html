Política de Privacidade do SinaluzPipeline

Última atualização: 16 de julho de 2026

1. Informações Coletadas

O SinaluzPipeline coleta apenas as informações necessárias para publicar conteúdo nas contas conectadas do usuário (TikTok, YouTube, Instagram), como tokens de autorização fornecidos pelas próprias plataformas durante o processo de login.

2. Uso das Informações

As informações são utilizadas exclusivamente para autenticar e publicar vídeos nas contas autorizadas pelo usuário. Nenhum dado é vendido, compartilhado ou utilizado para fins publicitários.

3. Armazenamento e Segurança

Os tokens de acesso são armazenados de forma segura e utilizados apenas pelo sistema de automação para as publicações autorizadas.

4. Compartilhamento com Terceiros

O SinaluzPipeline não compartilha dados pessoais com terceiros, exceto quando exigido pelas próprias plataformas conectadas (TikTok, YouTube, Instagram) para realizar a publicação do conteúdo.

5. Direitos do Usuário

O usuário pode revogar o acesso do SinaluzPipeline às suas contas a qualquer momento, diretamente nas configurações de aplicativos conectados de cada plataforma.

6. Contato

Dúvidas sobre esta política podem ser enviadas para sinaluz5@gmail.com.
